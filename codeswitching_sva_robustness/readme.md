The codeswitched directory studies how language mixing (codeswitching) affects neural circuits that handle Subject-Verb Agreement. Specifically, it:
Creates controlled codeswitching datasets with 4 levels of language mixing:
Pure Spanish (no switching)
Spanish with English nouns mixed in
Spanish with English verbs mixed in
Spanish with both English nouns and verbs mixed in
Analyzes how this affects model representations using PCA on attention head activations
Tests robustness by creating grammatically corrupted versions
Measures interference effects on grammatical processing
